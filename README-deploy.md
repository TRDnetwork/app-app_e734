# Deploy Instructions: Dawn Brew Coming Soon

## Deploy to Vercel

1. Push code to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click "New Project" and import your repository
4. Vercel will auto-detect this as a static HTML site
5. Click "Deploy" — no build command or output directory needed
6. Once deployed, set a custom domain (optional)

## Environment Variables

This app has no environment variables. All functionality is static and client-side only.

## First-time setup

No setup required. This is a fully static site with:
- No database
- No authentication
- No API integrations
- No email delivery

All features (including email form success message) are UI-only with fake feedback.
