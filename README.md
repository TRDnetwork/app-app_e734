# Dawn Brew – Coming Soon Landing Page

**Hand-roasted in Karnal, opening Summer 2026**

A minimalist, static "Coming Soon" landing page for *Dawn Brew*, an artisan coffee shop set to open in Karnal. This site focuses on brand storytelling, anticipation, and email sign-up — all with a warm, dark aesthetic evoking early morning light and craft roasting.

Built with simplicity in mind: plain HTML, CSS, and minimal JavaScript. No backend, no database, no authentication.
